# Ionic 3 example
# Io-AccuWeather by [Pavel Valentov](http:/valentov.ru)

Before Installation:
```bash
NPM — 5.3.0
Node.js — 8.2.1
Ionic — 3.9.2
Cordova — 7.0.1
Angular-CLI — 1.3.2
```

Install application:
```bash
$ npm install -g cordova ionic
$ npm install
```

Start application:
```bash
$ ionic serve
```
or
```bash
$ ionic cordova run browser
```
or
```bash
$ ionic cordova run android --prod --release
```
